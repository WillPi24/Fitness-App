Bring your training with you

Export from your old app, drop the file into Helm, and everything maps to the right place. No lost history, no re-entry.

How it works

Three steps. No source-app picker.

1

Export your data

Use your current app's own export feature. Helm reads standard CSV exports from the apps listed below, plus its own JSON backup file. Each app handles export differently - see their help page for the current steps.

2

Drop the file into Helm

Open Helm → Account → Import Data → choose your file. Helm detects the format automatically, so you don't need to tell it which app the file came from.

3

Review and confirm

Helm maps exercises, units, and dates into your library and shows a preview before anything is saved. Duplicates are flagged and unrecognised entries pass through as-is.

Under the hood

What Helm does with your file.

Auto-format detection

No "select your source" dropdown. Helm inspects the file header and picks the right parser automatically.

Smart exercise mapping

Over 150 exercise name variations across the supported apps fold into a single Helm entry, so your PR history stays intact.

Unit detection

Kilograms vs pounds is read from the file where possible. Helm only asks when the source file is ambiguous.

Duplicate protection

Workouts, runs, and meals with the same signature aren't re-imported, so you can safely import the same file twice.

FAQ

Common questions

What if my app isn't in the list?

A CSV with Date, Exercise, Reps, and Weight columns usually works for strength. Helm also accepts its own JSON backup, so if you can persuade your current tool to produce a close-enough CSV, Helm will try to import it.

What if my export is missing some fields?

Helm imports whatever is present in the file. Fields that aren't in the export just stay blank - they won't block the import. You can always fill things in later.

What happens to exercises Helm doesn't recognise?

They're imported under the exact name from the source file. You can rename, merge, or hide them later from your exercise library.

Can I import the same file more than once?

Yes. Helm detects duplicates by matching the signature of each workout, run, or meal, so a second import of the same file won't duplicate your history.
